🔧 TAPD 批量导入导出功能（来源：memory/2026-07-15.md）#

  • 导出脚本: tapd_browser/batch_export_cases.py - 创建任务→轮询→下载
  • 导入脚本: tapd_browser/batch_import_cases.py - 上传→创建任务→轮询
  • 公共模块: tapd_browser/batch_cli_common.py - Cookie 管理、轮询、公共参数
  • 服务端接口: 新增 POST /api/external/tapd-browser-cookie/get，使用密码 dev-admin-api-key-12345 进行外部鉴权
  • 技术要点:
    • PUT 上传不带 TAPD Cookie 会被网关拦截（WAF 403），必须在会话中携带 Cookie
    • 文件上传接口返回的是 UploadUrl/FilePath，而非预期的 Url/Key
    • 下载地址在任务结果的 ResultFileUrl（fs-data-gateway），不是灯塔统计（otheve.beacon.qq.com）

⚠️ 关键教训#

  • Python 注释风格错误: 发现并修复了 server_cookie.py 中的 JavaScript 风格注释（/** */），应使用 Python 三引号注释（# """）
  • GitLab 保护分支限制: main 分支受保护，不允许 force push，需要合并无关历史后普通推送
  • WAF 网关拦截: PUT 上传不带 TAPD Cookie 会被网关拦截，必须在会话中携带 Cookie
  • 字段识别陷阱: TAPD 上传接口返回的是 UploadUrl/FilePath，而非预期的 Url/Key

  • TAPD 链接格式: 统一从 /tapd_fe/ 改为 /tapd/（去掉 _fe）
