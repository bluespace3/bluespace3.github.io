Playwright MCP Server 配置

在 Claude Code 的配置文件中添加 MCP 服务器配置：

{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-playwright"
      ]
    }
  }
}

或者在 .claude.json 中配置：

"playwright": {
  "type": "stdio",
  "command": "npx",
  "args": [
    "-y",
    "@modelcontextprotocol/server-playwright"
  ],
  "env": {}
}
