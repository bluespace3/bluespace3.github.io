技术探索#

1. sed 高级脱敏技巧#

脱敏规则示例：

# 手机号(11位,1开头)
1[3-9][0-9]{9} → [手机号已脱敏]

# IP地址(x.x.x.x)
[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3} → [IP已脱敏]

# API Key(sk-开头)
sk-[a-zA-Z0-9_\-]{20,} → [密钥已脱敏]

# 复杂格式Token
[a-fA-F0-9]{32,}\.[a-zA-Z0-9_\-]{10,} → [密钥已脱敏]
[a-fA-F0-9]{40,64} → [密钥已脱敏]
[a-zA-Z0-9_\-]{40,100} → [密钥已脱敏]

最佳实践：

  1. 先保护 UUID 和日期（使用占位符），最后恢复
  2. 使用单词边界 \b 避免误匹配
  3. 精确匹配，避免过度脱敏
  4. 用真实数据测试脱敏效果
