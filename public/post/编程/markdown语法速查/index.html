Markdown 常用语法笔记#

Markdown 是一种轻量级标记语言，用纯文本格式编写，可转换成结构化的 HTML 文档。以下为常用语法整理，便于日常写作与查阅。

1. 标题#

在行首使用 # 号加空格，# 的数量代表标题级别（1–6级）。

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

效果预览：

一级标题#

二级标题#

三级标题#

2. 强调（粗体、斜体、删除线）#

语法 效果
**粗体** 或 __粗体__ 粗体
*斜体* 或 _斜体_ 斜体
***粗斜体*** 粗斜体
~~删除线~~ 删除线

3. 列表#

无序列表#

使用 -、* 或 + 加空格。

- 项目一
- 项目二
  - 嵌套子项（缩进两个空格或一个制表符）
* 也可以用星号
+ 也可以用加号

有序列表#

数字加点加空格，序号会自动递增。

1. 第一项
2. 第二项
   1. 子项（缩进）
3. 第三项

5. 代码#

行内代码#

用反引号包裹：code

代码块#

用三个反引号（或四个空格缩进）包裹，并指定语言以高亮。

```python
def hello():
    print("Hello, Markdown!")


**效果：**

```python

    print("Hello, Markdown!")

6. 引用#

在行首使用 > 加空格。

> 这是一段引用。
> 可以多行连续。
> 
> > 嵌套引用。

效果：

这是一段引用。 可以多行连续。

嵌套引用。

7. 表格#

用 | 分隔列，用 - 分隔表头与内容，冒号可指定对齐方式。

| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 内容   | 内容     | 内容   |
| 行2    | 行2      | 行2    |

效果：

左对齐 居中对齐 右对齐
内容 内容 内容
行2 行2 行2

8. 分割线#

三个或更多的 -、* 或 _ 单独一行。

---
***
___

效果：（以下为一条水平线）

9. 脚注（扩展语法，部分平台支持）#

在引用处写 [^标识]，在文末定义脚注内容。

这里需要注释[^1]。

[^1]: 这是脚注的具体内容。

10. 任务列表（扩展语法，GitHub 等支持）#

使用 - [ ] 表示未完成，- [x] 表示已完成。

- [x] 完成笔记编写
- [ ] 复习语法
- [ ] 实践应用

效果：

  • 完成笔记编写
  • 复习语法
  • 实践应用

11. 转义字符#

用反斜杠 \ 转义特殊符号，使其原样显示。

\* 星号不会被解析为强调 \*
\# 井号不会被解析为标题

效果： 星号不会被解析为强调

12. 内嵌 HTML#

Markdown 支持直接书写 HTML 标签，实现更复杂的排版。

<p align="center">这是一段居中的文字。</p>

提示：不同平台（如 GitHub、Typora、Notion、Obsidian）对 Markdown 的支持略有差异，扩展语法需确认环境是否兼容。

这份笔记已涵盖日常写作所需的绝大部分 Markdown 语法，可保存备用。
