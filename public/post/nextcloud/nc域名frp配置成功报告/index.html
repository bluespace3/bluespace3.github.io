📊 性能分析#

为什么两者延迟接近？#

Tailscale + Cloudflare (1.40秒):

  • Cloudflare CDN: 全球节点，可能跳转多次
  • TLS 握手: 多次加密/解密
  • Tailscale: WireGuard 加密
  • 总开销: CDN + TLS + VPN

FRP (1.39秒):

  • 直连服务器
  • FRP 隧道转发
  • TLS 握手
  • 总开销: 隧道 + TLS

结论:

  • ✅ FRP 没有比 Tailscale 慢很多
  • ✅ 两者性能相当
  • ✅ Cloudflare CDN 在这个场景下优势不明显
