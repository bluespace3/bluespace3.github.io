## 🔍 技术分析

### Tailscale 为什么快？

1. **直连路径**
   ```
   Mac → 服务器 (xxx.xxx.xxx.xxx)
   单跳，延迟: 13.86ms
   ```

2. **WireGuard 加密**
   - 轻量级 VPN 协议
   - 加密开销小
   - 内核级优化

3. **NAT 穿透**
   - 自动建立最优路径
   - 避免中转服务器

### FRP 为什么慢？

1. **多跳转发**
   ```
   Mac → FRP 服务器 → Mac (容器)
   两跳，延迟: 1176.93ms
   ```

2. **额外开销**
   - TCP 隧道封装
   - 多次网络跳转
   - 服务器处理延迟

3. **可能的瓶颈**
   - FRP 服务器性能
   - 网络质量
   - 容器端口映射
