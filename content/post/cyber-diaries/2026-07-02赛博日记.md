# 2026-07-02 赛博日记

## 核心工作

### 🔧 本地服务修复：Nextcloud + Colima
- **问题**: 文件管理系统和 Nextcloud 无法访问
- **排查**: 文件管理系统（端口 8003）实际在运行；Nextcloud 因 Colima 未启动导致全部 Docker 容器挂掉
- **根因**: Colima 没有开机自启 → Docker 引擎没跑 → 三个容器全挂；同时 Docker 虚拟磁盘满了（97.9G 用了 94.8G），MySQL 因磁盘空间不足反复崩溃
- **修复**: 
  - 清理 Docker 无用资源释放 25.6GB
  - 重启 MySQL → Nextcloud → Redis 全线恢复
  - 创建 `~/Library/LaunchAgents/com.colima.autostart.plist` 实现开机自启
- **状态**: ✅ 全部恢复

### 🌐 域名访问排查：mm.skyspace.eu.org
- **问题**: OpenClaw 外网域名无法访问
- **排查链路**: 
  - OpenClaw 本地服务 (18789) ✅
  - LAN 访问 (xxx.xxx.xxx.xxx:18789) ✅
  - DNS → Cloudflare ✅
  - Cloudflare TLS ✅
  - Cloudflare → 源站 ❌ 链路不通
- **深入**: 服务器 nginx 反代到 Tailscale IP `xxx.xxx.xxx.xxx:18789`，但 Tailscale 隧道实际是断的（`tx 21528 rx 0`，Mac 端 offline 26 分钟）
- **结论**: Tailscale 隧道断裂导致服务器无法转发请求到 Mac
