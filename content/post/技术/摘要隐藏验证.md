## 总结

本文章用于验证：

1. ✅ **文章列表中**：应该只显示 `<!--more-->` 之前的内容，加密内容完全不可见
2. ✅ **文章详情页**：可以看到完整的加密锁定界面
3. ✅ **密码验证**：正确输入密码后可以看到加密内容
4. ✅ **多加密区域**：支持同一文章中的多个加密区域

**使用方法：**

- 在文章中使用 `{{</* encrypted password="你的密码" */>}}` 开始加密内容
- 使用 `{{</* /encrypted */>}}` 结束加密内容
- 使用 `<!--more-->` 分隔符控制摘要显示范围

**最佳实践：**

1. 始终使用 `<!--more-->` 分隔符来控制摘要
2. 将敏感内容放在加密区域内
3. 确保在文章中提供足够的有价值的公开内容作为摘要
4. 使用强密码保护敏感信息
