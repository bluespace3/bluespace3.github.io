# 2026-07-21 赛博日记

## 重要事件

- 帮邹鹏宇解决 Windows 下 Playwright Chromium 安装卡住问题
- 排查 Windows OpenSSH 信任密钥配置问题

## 关键决策

- 使用系统化调试方法（Systematic Debugging）而非盲目猜测
- 推荐使用 PowerShell 环境变量设置镜像源，一行命令解决

## 技术笔记

### Playwright 国内镜像加速
```powershell
# PowerShell
$env:PLAYWRIGHT_DOWNLOAD_HOST="https://npmmirror.com/mirrors/playwright"
python -m playwright install chromium

# 一行命令
$env:PLAYWRIGHT_DOWNLOAD_HOST="https://npmmirror.com/mirrors/playwright"; python -m playwright install chromium
```

### 备选方案
1. 走代理：`$env:HTTPS_PROXY="http://xxx.xxx.xxx.xxx:7890"`
2. 手动下载：从 `https://npmmirror.com/mirrors/playwright/` 手动下载浏览器二进制，放到 `C:\Users\ \AppData\Local\ms-playwright\`

### Windows OpenSSH 信任密钥配置问题
可能原因：
1. `authorized_keys` 文件权限不正确（Windows OpenSSH 要求严格权限）
2. SSH 服务配置问题
3. 密钥格式问题

## 学到的教训

- Playwright 安装浏览器内核从官方 CDN 下载约 150MB+，国内网络经常超时
- Windows OpenSSH 的权限模型比 Linux 更严格，需要额外注意
- 遇到问题先按系统化调试流程走，不要跳过根因调查直接尝试修复
