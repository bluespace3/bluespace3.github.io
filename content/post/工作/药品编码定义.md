| 药品通用名（如“阿司匹林肠溶片”）                          | 药品本体 / 药典条目     | 不是商品编码                                               |
| ----------------------------------------------------------- | ----------------------- | ---------------------------------------------------------- |
| 药品通用名 + 规格 + 剂型                                    | SPU（业务定义）         | 可作为产品系列，但药品行业常将厂家纳入 SPU                 |
| **品牌 + 药品通用名 + 规格 + 剂型 **                       | upc                     | 最小可售、可库存单位                                       |
| **品牌 + 药品通用名 + 规格 + 剂型 + 厂家 + 包装数量** | **SKU（更精确）** | 若同一厂家同规格有不同包装（如10片装 vs 20片装），则需区分 |
| 每个 SKU 对应的条形码                                       | UPC / EAN               | 通常一一对应                                               |
