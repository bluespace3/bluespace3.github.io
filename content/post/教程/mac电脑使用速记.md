在Mac终端中，有几种方法可以连续删除命令：

1. 使用 Ctrl + W ：删除光标前的一个单词
2. 使用 Ctrl + U ：删除光标前的整行内容
3. 使用 Ctrl + K ：删除光标后的整行内容
4. 使用 Option + Delete ：删除光标前的一个单词（与 Ctrl + W 类似）
   这些快捷键可以帮助你快速删除命令内容，提高终端操作效率。
