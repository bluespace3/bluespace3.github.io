## terminus快捷键使用合集

| 功能                 | macOS                         | Windows / Linux                   |
| -------------------- | ----------------------------- | --------------------------------- |
| 新建标签页           | `⌘ + T`                      | `Ctrl + T`                        |
| 关闭当前标签页       | `⌘ + W`                      | `Ctrl + W`                        |
| 切换标签页（1-8）    | `⌘ + 1~8`                    | `Ctrl + 1~8`                      |
| 切换到最后标签页     | `⌘ + 9`                      | `Ctrl + 9`                        |
| 下一/上一标签页      | `⌘ + Tab`/`⌘ + Shift + Tab` | `Ctrl + Tab`/`Ctrl + Shift + Tab` |
| 全屏                 | `⌘ + Ctrl + F`               | `F11`                             |
| 复制（需先选中文本） | `⌘ + C`                      | `Ctrl + C`                        |
| 粘贴                 | `⌘ + V`                      | `Ctrl + V`                        |
| 全选                 | `⌘ + A`                      | `Ctrl + A`                        |
| 查找文本             | `⌘ + F`                      | `Ctrl + F`                        |
| 打开设置             | `⌘ + ,`                      | `Ctrl + ,`                        |
| 重连当前会话         | `⌘ + R`                      | `Ctrl + R`                        |
| 打开会话侧边栏       | `⌘ + B`                      | `Ctrl + B                         |
| `                    |                               |                                   |

终端内

| 类别     | 快捷键                    | 作用                                         |
| -------- | ------------------------- | -------------------------------------------- |
| 光标移动 | `Ctrl + A`/`E`            | 行首 / 行尾                                  |
|          | `Ctrl + B`/`F`            | 左/右移1字符                                 |
|          | `Alt + B`/`F`             | 左/右移1单词                                 |
| 编辑删除 | `Ctrl + U`                | 删除光标→行首                               |
|          | `Ctrl + K`                | 删除光标→行尾                               |
|          | `Ctrl + W`                | 删除前一个单词                               |
|          | `Ctrl + Y`                | 粘贴（yank）刚删除的内容（Shell 内部剪贴板） |
| 历史操作 | `Ctrl + R`                | 反向搜索历史命令                             |
|          | `↑`/`↓`或`Ctrl + P`/`N` | 上/下一条历史                                |
| 其他     | `Tab`                     | 自动补全命令/路径                            |
|          | `Ctrl + L`                | 清屏（等效`clear`）                          |
|          | `Ctrl + C`                | 中断当前进程（⚠️ 非复制！）                |
|          | `Ctrl + Z`                | 挂起进程（用`fg`恢复）                       |
|          | `Ctrl + D`                | 删除字符 / 退出 Shell（EOF）                 |
